# 🤖 AI Agents 201 Project

## 🚀 Overview
As part of the AI Agents 201 Workshop, I built this project to explore how AI agents can solve real-world problems using automation and APIs.

This project includes multiple agents that can fetch real-time data like time and weather, and also work together as a multi-agent system.

---

## 🔹 What I Built
- ⏰ Time Agent – Returns the current time based on location  
- 🌦️ Weather Agent – Fetches live weather data using OpenWeather API  
- 🤝 Multi-Agent System – Combines multiple agents to handle tasks intelligently  

---

## 🛠️ Tech Stack
- Python  
- Google ADK  
- OpenWeather API  
- pytz  

---

## 📂 Project Structure

- weather_agent/
